# Terraria 役次元 IM 联动

基于 IM 链路的 Terraria 与役次元事件联动 Mod。

## 功能简介

- 采集 Terraria 内关键事件
- 将事件转发为发送给役次元的 `WebSocket / IM` 指令
- 适合把游戏事件接到你现有的役次元联动链路

## 当前支持事件

- 玩家受伤、死亡、复活
- 被 NPC / 投射物命中
- 物品拾取
- Boss 出现、Boss 击败
- 白天开始、夜晚开始
- 血月、下雨、史莱姆雨、日食
- 入侵开始、入侵结束

## 输出方式

1. `websocket_command`
   对接发送到役次元的 WebSocket / IM 指令协议

## 使用说明

- 先准备一个可与役次元联动的下游服务
- 再在本 Mod 的 `settings.json` 与 `routes.json` 中配置联动规则
- 进入世界后可直接用图形化配置界面发送测试事件验证链路

## 说明

本 Mod 当前重点负责“Terraria 事件采集 + 规则路由 + 役次元 IM 指令转发”。
旧版蓝牙连接与波形触发能力已经从 Mod 中移除。
